## Rodando o Sistema

###  1. BMVC pelo BASH:

#### Requísitos:
1. Python 3.x
2. Virtual Environment (venv)
3. Bottle (pip install bottle)

#### Rodando pelo BASH

1. Executar o arquivo 'route.py', na raiz do projeto.

### 2. BMVC pelo DOCKER

#### Baixando o Docker:

Página oficial do [Docker](https://docs.docker.com/engine/install/).

Como instalar o Docker no Arch Linux (pela gracinha)

#### Montando pelo DOCKER:

1. $ docker build -t bmvci .
2. $ docker run -d -p 8080:8080 -v $(pwd):/app bmvci
